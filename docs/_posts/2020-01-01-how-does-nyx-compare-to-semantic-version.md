[*semantic-release*](https://github.com/semantic-release/semantic-release) is an excellent tool and if you're working on Node projects you should go that way. However, when not on Node projects, it introduces some constraints that are not always so simple to overcome. *Nyx* is an alternative to *semantic-release* for those cases.

See [here]({{ site.baseurl }}{% link _pages/guide/user/08.transitioning/from-semantic-release.md %}) for details on the differences.
