## How to read this guide

In order to reduce clutter, repeating the documentation for different configuration means, this guide shrinks overall informations into tables, where each configurable options has one column for each way of configuring it. More precisely, for each option:

* command line option names are explained in the *Command Line Option*
* environment variable names are explained in the *Environment Variable*
* option names used in configuration files are explained in the *Configuration File Option*

However, since several different grammars are available for configuration files, each option must be represented in a specific way, as further explained in the [following section](#supported-configuration-file-grammars).
