This section gives you detailed reference information about the available [standard configurations]({{ site.baseurl }}{% link _pages/guide/user/02.introduction/standard-configurations.md %}) that come preconfigured with Nyx. You don't have to use one, but if you do you can just save time in dealing with your own configuration.

Remember you can use them by using the [`preset`]({{ site.baseurl }}{% link _pages/guide/user/03.configuration-reference/global-options.md %}#preset) option. When in doubt start considering the [simple]({{ site.baseurl }}{% link _pages/guide/user/04.configuration-presets/simple.md %}) preset.
