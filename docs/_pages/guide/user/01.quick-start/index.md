In this section you can find instructions to get started in minutes depending on your specific use case and platform.

## Presets

You can enable configuration [presets]({{ site.baseurl }}/reference/standard-configurations/) that come with Nyx out of the box to enable complete configuration sets with just one flag.

These streamlined configurations can cover all needs of most teams and are maintained along with the project to relief you from complex configurations.

## Using plugins

If you are using [Gradle](https://gradle.org/) as your build tool you can jump right to the [Gradle Plugin page]({{ site.baseurl }}/quick-start/gradle-plugin/).

## Using libraries

In case you are interested in developing your own project instead of using Nyx you can check out the [Java Library page]({{ site.baseurl }}/java-library/).
