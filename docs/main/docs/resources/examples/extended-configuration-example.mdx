# Extended configuration example

Here you can find the example of a large configuration in the supported [`formats`](../../user/introduction/configuration-methods.mdx#supported-file-grammars).

This configuration doesn't use any [preset](../../user/configuration-presets/index.md) but sets many [global configuration options](../../user/configuration-reference/global-options.mdx) explicitly along with [commit message conventions](../../user/configuration-reference/commit-message-conventions.mdx) and [release types](../../user/configuration-reference/release-types.mdx). Where allowed, [templates](../../user/configuration-reference/templates.mdx) are also used.
